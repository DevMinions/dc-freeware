# Dc · 工业数据采集（免费版 Freeware）

> 通用工业数据采集客户端，内置**本地 AI 副驾**。免费下载使用，闭源二进制发行。

面向工业现场的数据采集软件：浏览设备地址空间、配置采集任务、实时订阅并经 TCP 发布到下游。全套 AI 能力**在本地运行**（数据不出厂），无需联网、无需云端。

## ✨ 亮点

- **多协议接入** — OPC UA · Modbus · Siemens S7
- **本地 AI 副驾** — 自然语言建任务/取点/设缩放/告警合理化、证书与运行期排错助手、运维问答（RAG over 实时态），全部本地大模型推理
- **AI × 动效融合** — 数据流动可视化、AI 扫描诊断、答案接地高亮、建议预演
- **AI 模型管理** — GUI 内热切换模型、GPU（Vulkan）加速开关、推理统计
- **两种形态** — Windows 桌面端 GUI + 无头网关（Web 控制台）
